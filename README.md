# PDF 서명 & 편집기 (+ 오피스 뷰어)

PDF에 **서명·텍스트·날짜·체크**를 올려 저장하고, **Word·Excel·PowerPoint·이미지는 미리보기**로 여는 도구입니다.
**여러 파일을 동시에 열어** 왼쪽 사이드바에서 전환할 수 있습니다.
모든 처리는 사용자의 브라우저 안에서만 이뤄지며, 파일은 외부로 전송되지 않습니다.

## 실행 방법 (3가지 중 택1)

| 방법 | 파일 | 설명 |
|------|------|------|
| ⭐ **앱 실행파일** | `pdf-signer.exe` | 더블클릭 → 로컬 서버가 뜨고 브라우저가 자동으로 열림. **설치·인터넷 불필요.** |
| **오프라인 HTML** | `pdf-signer-offline.html` | 더블클릭 → 브라우저에서 바로 열림. 라이브러리 내장이라 **인터넷 불필요.** |
| **온라인 HTML** | `pdf-signer.html` | 가장 가벼운 원본. 라이브러리를 CDN에서 받으므로 **첫 실행 시 인터넷 필요.** |

> 셋 다 기능 동일. 배포·공유용이면 `pdf-signer.exe` 또는 `pdf-signer-offline.html` 하나만 주면 됩니다.
> **PowerPoint는 `pdf-signer.exe`에서 가장 안정적으로 표시됩니다.**

## 지원 형식

| 형식 | 확장자 | 동작 |
|------|--------|------|
| PDF | `.pdf` | **서명·편집·저장** (드래그로 요소 배치 → 다운로드) |
| Word | `.docx` | 미리보기 (**열기 암호 지원**) |
| Excel | `.xlsx` `.xls` `.csv` | 미리보기 (시트 탭 전환, **열기 암호 지원**) |
| PowerPoint | `.pptx` | 미리보기 (레이아웃 근사치, **열기 암호 지원**) |
| 이미지 | `.png` `.jpg` `.gif` `.webp` `.bmp` `.svg` `.avif` `.ico` | 미리보기 (확대/축소) |

## 사용법

> 🗂 **여러 파일 동시 열기** — 왼쪽 **사이드바**가 열린 파일 목록입니다. 클릭으로 전환, × 로 닫기, **＋ 파일 추가**로 계속 추가. 상단 툴바는 **선택한 파일에 맞춰 자동 전환**됩니다(PDF=서명·다운로드 / 그 외=인쇄).

1. 파일을 화면에 **드래그&드롭** (또는 클릭/＋ 파일 추가) — 한 번에 여러 개도 가능
2. **PDF면 편집 모드:**
   - ✍️ 서명(그리기 / 🖼 이미지 업로드 / 최근 서명 재사용) · 🔤 텍스트 · 📅 날짜 · ✓ 체크
   - 드래그(이동) · 더블클릭(텍스트 수정) · 모서리(크기) · `Delete`(삭제) · 선택 시 글자 크기/색상
   - **⬇️ 다운로드** → `원본이름_signed.pdf` 저장 (원본은 그대로 둠)
3. **Word·Excel·PowerPoint·이미지면 미리보기 모드:**
   - 내용을 화면에서 확인. **🖨 인쇄 / PDF로 저장** 버튼으로 브라우저 인쇄→PDF 저장 가능
   - 이미지는 클릭, 또는 하단 −/+/⤢ 버튼으로 확대·축소
   - 🔒 암호 걸린 Office 파일은 열 때 **암호 입력창**이 떠요 (입력하면 그 자리에서 복호화)
   - 📂 새 파일로 다른 파일 열기

## 다시 빌드하려면

소스(`pdf-signer.html`)를 수정한 뒤:

```bat
node build-offline.js     :: 1) 오프라인 HTML 재생성 (Node 필요)
cd desktop & build.bat     :: 2) exe 재생성 (Go 필요)
```

- 오프라인 빌드는 `vendor/`의 모든 라이브러리를 HTML에 인라인합니다.
- exe 빌드는 `pdf-signer-offline.html`을 `desktop/app.html`로 복사해 바이너리에 내장합니다.

## 파일 구조

```
pdf-signer.html           ← 원본 소스 (CDN, 사람이 읽기 좋은 버전)
pdf-signer-offline.html   ← 빌드 산출물: 라이브러리 내장 단일 HTML (오프라인, ~3.3MB)
pdf-signer.exe            ← 빌드 산출물: 로컬 서버 앱 (오프라인 HTML 내장, ~9MB)
build-offline.js          ← 오프라인 HTML 빌드 스크립트 (Node)
vendor/                   ← 라이브러리 모음 (오프라인 빌드 재료)
desktop/                  ← Go 앱 소스 (main.go, build.bat 등)
```

## 사용 기술

- **PDF.js** (렌더) + **pdf-lib** (저장) — PDF 서명/편집
- **docx-preview** — Word, **SheetJS(xlsx)** — Excel, **PPTXjs**(+jQuery·JSZip) — PowerPoint
- 열기 암호 복호화: **crypto-js** 기반 자체 모듈 `vendor/office-decrypt.js` (ECMA-376 Agile) — Word·Excel·PPT 공통, 브라우저 안에서 복호화
- **Go** — 오프라인 HTML을 내장해 로컬 서버로 제공하는 단일 실행파일

## 알아두실 점 / 한계

- **오피스 파일은 보기 전용**입니다(서명은 PDF에서). 필요하면 PDF로 변환하는 기능을 추가할 수 있어요.
- **PowerPoint 미리보기는 근사치**입니다 — 복잡한 레이아웃·애니메이션은 어긋날 수 있고, **차트는 표시되지 않습니다.**
- **암호 걸린 Word·Excel·PowerPoint(.docx/.xlsx/.pptx)** 는 암호를 입력하면 열립니다(최신 Agile 암호화 기준, 약 1~3초 소요). 단 구형 Standard 암호화, 레거시 이진(`.doc` `.ppt`, 암호 `.xls`), **암호 걸린 PDF**는 아직 미지원입니다.
- PDF에 추가한 텍스트는 저장 시 이미지로 들어가 선택/검색이 안 됩니다(서명·기입용으로는 일반적인 방식).
- 회전된 PDF 페이지(/Rotate ≠ 0)는 요소 위치가 약간 어긋날 수 있습니다(경고 표시).
- 인증서 기반 전자서명(PKI)이 아니라 **눈에 보이는 서명**입니다.
